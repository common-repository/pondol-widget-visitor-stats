Visitors statistics for daily, monthly, weekly and per year.
Visitors's reference be shown.


== Installation ==
1. extract pondolwidget_visitorstats.zip and Upload  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widget
4. Drag and Drop where you display this plugin.
